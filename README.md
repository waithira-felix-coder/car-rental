### SwiftRide Rentals Website

A fully responsive car rental website built with **HTML, CSS, and JavaScript**.  
The site provides an intuitive user experience with sections for Home, Rent, Ride, Gallery, Stories, Contact, and Find Location.  
It is designed to be professional, mobile-friendly, and easy to navigate.

---

##  Features

- **Responsive Design**: Works seamlessly across desktops, tablets, and mobile devices.
- **Modern UI/UX**: Clean layout with clear navigation (Home, Rent, Ride, Gallery, Stories, Contact, Find Location).
- **Car Gallery**: Showcases available vehicles with images and specifications.
- **Stories Behind the Wheel**: Blog-style section with 3 posts highlighting experiences and updates.
- **Find Location**: Integrated map visuals to help users locate rental offices easily.
- **Subscribe Button**: Connected via **FormSpree** to deliver subscription requests directly to the owner’s email.
- **Contact Section**: Simple form and contact details for customer support.

---

##  Technologies Used

- **HTML5** – Structure and semantic markup
- **CSS3** – Styling, responsive layouts, animations
- **JavaScript (ES6+)** – Interactivity, dynamic content handling
- **FormSpree** – Email delivery for subscription form submissions
